A virtual development environment using veewee, vagrant and virtualbox.

Currently including 

 * erlang
 * emacs
 * git
 * mininet (Virtual network, http://mininet.org)

## Get started

### Prereq's

* bundler
* ruby 1.9.3+
* virtualbox
* vagrant 1.2.x (http://downloads.vagrantup.com)

### Installation 

	$ boxxu.sh build

### Start

	$ boxxu.sh start

### Stop
	
	$ boxxu.sh stop
