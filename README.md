A devbox using veewee, vagrant and virtualbox

## Get started

### Prereq's

* bundler
* ruby 1.9.3+
* virtualbox

### Installation 

	$ boxxu.sh build

### Start

	$ boxxu.sh start

### Stop
	
	$ boxxu.sh stop

## Version Notes

* 0.0.1 - based on veewee's vbox template 'ubuntu-12.10-server-amd64'
